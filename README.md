Visualização de Dados de Ecommerce

Projeto desenvolvido na formação de Analista de Dados da EBAC.

Objetivo

Criar gráficos a partir do arquivo `ecommerce_estatistica.csv`, utilizando Python para analisar padrões, relações e distribuições dos dados.

Gráficos criados

* Histograma
* Gráfico de dispersão
* Mapa de calor
* Gráfico de barras
* Gráfico de pizza
* Gráfico de densidade
* Gráfico de regressão

Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab / Jupyter Notebook
* GitHub

Arquivos

* `ecommerce_estatistica.csv` — base de dados utilizada
* `visualizacao_ecommerce.ipynb` — notebook com análise e gráficos
